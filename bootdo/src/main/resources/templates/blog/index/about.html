关于我

BootDo 是一个面向学习型的开源框架，采用 Spring Boot + MyBatis + Thymeleaf + Shiro 技术栈开发。我们的目标是提供一个简洁高效、代码透明的学习平台，帮助开发者快速理解和掌握现代 Web 开发技术。

我们相信，优秀的代码应该简洁、直观、易于理解。通过减少过度封装，展现技术本质，让学习者能够看到最原始、最纯粹的代码实现。
